Scope Information for "mysample"

Styles represent containers for per-engine scopes to be applied to matched text. Iro requires that every piece of matched text that is matched is assigned to an internal style. In turn, these styles can be mapped on a one-to-one basis with the scope representations of various exporters.

Whilst it is not a requirement to fully map to all exporters, it is strongly encouraged to define a full set of mappings for all exporters. This screen highlights where there is a gap in coverage, or perhaps where there is ambiguity.
